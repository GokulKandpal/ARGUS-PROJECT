<h2>📌 Project Overview</h2>

ARGUS is a surveillance and intelligence platform developed by INNEFU that provides monitoring, analytics, alert handling, search operations, investigation workflows, and case management capabilities.<br>
This project focuses on validating application functionality and end-to-end workflows across multiple modules using manual and automation testing approaches.<br><br>

<h2>🎯 Objective</h2>

The objective of this project is to verify that all modules function according to business requirements and ensure application quality through systematic testing.<br><br>

✔ Reports Module<br>
